# Documentation

This `docs/` directory is for human-friendly explanations of the repository.

It is organized by area of the codebase so you can quickly find the part you
want to learn about.

## Sections

- [Web App](./web/README.md): beginner-friendly notes about the Next.js app in
  `apps/web`.

## Why This Exists

Code tells the computer what to do. Documentation helps people understand why
the code is here, how it is organized, and where to make changes.
